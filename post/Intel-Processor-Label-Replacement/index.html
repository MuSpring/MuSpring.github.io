白嫖Intel处理器贴纸接生级教程

2023-04-27 Views 技巧750字3 min read繁

不说废话，直接上教程，无图，意义党退散。

一、前期准备

1.准备File Attachments

打开这个链接，下载Intel System Support Utility

下载后打开，直接点击Scan，到达Review后点击Next，再点击Save保存文本文档。

2.准备Header Editor

会科学上网的可以跳过。

打开这个链接安装即可。

打开Header Editor配置页面，点击导入和导出，在下载规则输入框处键入如下链接（二选一）。

https://azurezeng.github.io/static/HE-GoogleRedirect.json

https://www.azurezeng.com/static/HE-GoogleRedirect.json

感谢Github@AzureZeng

点击旁边的下载按钮，下载完成后点击保存。

二、开始申请

点击该链接

1.Name

不要输入全中文拼音，比如Yao Ying,请输入一个带有英文单词的名字，比如Potato Yao.

2.Street Address

将你的快递地址去掉省市翻译成英文放到这里。

不要学那些网上的美国式地址填法，快递是中国人送，这样只会添麻烦。

3.City与State

City是市，State是省。

直辖市两个填一样的。

4.Country

你猜你要选哪？

5.Postal Zip Code

全国邮政编码查询系统

查到了填在这里。

6.Contact Phone

+86 你的电话。

记得带空格。

7.Product Type Selection

Desktop 台式机，Laptop 笔记本。

按实际情况选。

8.Request Reason

写一篇小作文，告诉他们你多喜欢Intel的产品，你为什么要领取Intel的贴纸,最后感谢一下他们就可以。

9.File Attachments

点击browse files或者把你第一步保存的文本文档拖拽到这里。

10.E-mail Address

填邮箱地址。

QQ有概率能收到，163完全收不到，所以建议用Outlook或者Gmail.

11.I Agree

不点的你是故意找茬是吧？

12.Send me a copy of my responses

可选项，点了之后填上邮箱地址就可以收到提交回执。

回执邮件内容

Thank you! Your label request has been sent.

A email will be sent when request has been processed for shipping.

A copy is included below for your records.

三、后续

不出意外的话，你发送请求后的第一个美国工作日时间（北京时间约凌晨两点）英特尔就会受理。

受理邮件内容

标题:OLO (发货国家) - Label Request - (姓名)
Dear (姓名)

Thank you for wanting to obtain a replacement Intel® label for your system! Also, thank you for providing your proof of ownership/invoice/shipping address – we have validated this and will expedite an Intel® Core Processor replacement label to the address you provided.

下略.

一般情况下，你收到这封邮件后三日内就会安排顺丰特快空运从无锡送到你家。

快递单号请见你后续收到的邮件。

注意：部分情况下，你收到的快递单号顺丰快递官网无法查询，这时请尝试删掉前面的SF。

本人在小城市，三天到货。

如果你申请的贴纸很稀有，比如我之前申请的2013年奔腾贴纸，就会安排Fedex从Glasgow发货。

也很快，四天到货。
